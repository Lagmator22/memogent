# Memogent for iOS

This directory is a self-contained Swift Package that wraps Memogent's
stable C ABI and compiles the C++ core through SwiftPM.

## Add as a dependency

```swift
.package(url: "https://github.com/Lagmator22/memogent.git", branch: "main")
```

and in your target:

```swift
.target(name: "YourApp", dependencies: [.product(name: "Memogent", package: "memogent")])
```

## Usage

```swift
import Memogent

let engine = try Memogent.Orchestrator(dataDir: .documentsDirectory)
engine.recordAppOpen(appId: "com.foo.bar")
let predictions = engine.predictNextApps(k: 3)
engine.attachThermalHook()
```

## Running tests

```bash
cd bindings/ios
swift test
```

For production builds you should use the prebuilt `Memogent.xcframework`
generated by our CI pipeline — the in-package C++ sources are intended
for developer iteration and SwiftPM test runs.
